# Cashflow Tracker (Frontend)

Premium cash-management SaaS frontend built with Next.js (App Router), Tailwind CSS, and React Query. Designed for fast, accurate cash operations with a clean enterprise-grade UI.

## Tech Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- React Query
- Lucide Icons

## Key Features

- Period management (open/close with validation)
- Machine entry logging with difference breakdown
- Winners & payout tracking
- Business and user management
- Responsive, enterprise-grade UI

## Getting Started

```bash
npm install
npm run dev
```

Open the app at:

```
http://localhost:3000
```

## Environment

This frontend requires the backend API to be running. You can run the backend locally or use a development backend.

Backend repo:

```
https://github.com/sumitsapkota75/cashflow-tracker-api-java
```

Create a `.env` file with your API base URL. Example for local backend:

```
NEXT_PUBLIC_API_BASE_URL=http://localhost:8080
```

For development backend, use:

```
NEXT_PUBLIC_API_BASE_URL=https://cashflow-tracker-frontend.vercel.app
```
